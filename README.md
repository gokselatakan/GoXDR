# GoXDR

**KQL Query Library for Microsoft Sentinel and Defender XDR**

A curated collection of KQL detection and threat hunting queries ready to paste and run in Microsoft Sentinel and Defender XDR Advanced Hunting.

## What's Inside

- **KQL queries** covering identity security, email security, endpoint detection, OAuth governance, LDAP/NTLM attacks, SOC metrics and more
- **Searchable web interface** with category, severity and platform filters
- **Copy KQL** button for instant use in your environment
- **Dual platform support** — queries labeled for Sentinel, Defender XDR or both
- **Regularly updated** with new detections

## Live Site

**[https://gokselatakan.github.io/GoXDR/](https://gokselatakan.github.io/GoXDR/)**

## Categories

- Identity (Auth Attacks, MFA, Token Attacks, Geo Anomaly, Device Trust, AI Security)
- Non-Interactive Sign-In (Token Attacks, Shadow IT, Anomaly)
- NTLM (Attack Detection, Threat Intel, Compliance)
- LDAP (Attack Tools, Privileged Accounts, Reconnaissance, DC Performance)
- App Identity (Service Principals, Managed Identity, Botnet Detection)
- OAuth (Consent, Permissions, App Governance)
- Email (BEC, AiTM, Phishing, Malware, URL Protection, Data Exfiltration)
- Collaboration (Teams, SharePoint, OneDrive, Data Protection)
- XDR (Attack Chains, Investigation, Endpoint, SOC Metrics, Vulnerability Mgmt)
- Geolocation (Location Validation, Threat Correlation)

## Usage

1. Visit the [live site](https://gokselatakan.github.io/GoXDR/)
2. Search or filter by category, severity or platform
3. Click a query to see the description and KQL code
4. Click **Copy KQL** and paste into Advanced Hunting or Analytics Rule
