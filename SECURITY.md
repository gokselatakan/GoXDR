# Security Policy

## Overview

GoXDR is a static, open-source project that provides a collection of KQL queries for detection and threat hunting purposes.

This project **does not process, store, or transmit sensitive data**. All content is intentionally public and accessible via GitHub Pages.

---

## Supported Versions

As this is a continuously updated repository, only the latest version available in the `main` branch is considered supported.

---

## Reporting a Vulnerability

If you discover a potential security issue, please:

* Open an issue in this repository
* Clearly describe the finding, impact, and steps to reproduce

If the issue is sensitive, you may choose to limit public disclosure until it is reviewed.

---

## Scope

The following are **out of scope** for security vulnerabilities:

* Public availability of HTML, CSS, and JavaScript files (this is expected behavior)
* Downloading or cloning repository content
* Reuse or redistribution of KQL queries

---

## Security Considerations

* No secrets, API keys, or credentials should ever be committed to this repository
* Contributions are reviewed before being merged
* GitHub is used as the single source of truth

---

## Disclaimer

This project is provided "as is", without warranty of any kind. Users are responsible for validating and testing queries before using them in production environments.
